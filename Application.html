TA Apportionment Tool 1.0

Welcome to the webtool for apportioning teaching assisstants by Jaron Wang! This tool uses Jefferson's Method for apportioning a fixed number of TAs based on the student population at certain times. The project can be found online here, and this tool was created based upon the work TA Scheduling for the UW Introductory Programming Lab by Kaitlyn Arlt, Elizabeth McKinnie, and Jaron Wang for MATH381 under the instruction of Professor Sara Billey.
Note: App assumes that student populations are not tied. May overdistribute if ties exist.

Instructions:

  1. Enter the number of students at each time into the input table below.
  2. Enter the number of TAs available.
  3. Enter the number of hours each TA works.
  4. Click the Button.
